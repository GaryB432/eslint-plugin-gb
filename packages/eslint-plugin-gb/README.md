# eslint-plugin-gb

## Description

My favorite eslint configurations

## Installation

```
$ npm i --save-dev typescript @typescript-eslint/parser eslint-plugin-gb
```

## Usage

```js
// .eslintrc.js

module.exports = {
  parser: '@typescript-eslint/parser',
  plugins: ['gb'],
  extends: ["plugin:gb/recommended"],
};

```
